Aula 3.2

(CSS)

Caixas


Dimensões da Caixa


Por padrão, uma caixa recebe o tamanho exato para acomodar seu conteúdo. Podemos fixar estes valores utilizando as propriedades width e height. Para isso podemos fixar os valores em pixels ou percentagem.

Width: Define a largura do elemento.

width: 8%;

Height: Define a altura do elemento.

height: 8%;

Limitando largura e altura


Alguns designs de página se expandem e se contraem para se adaptar ao tamanho das telas, com estas propriedades podemo s limitar estas adaptações dando valores máximos e mínimos:

Min-width e max-width: Para definir mínimo e máximo para largura.

max-width: 8%;

Min-height e max-height: Para definir mínimo e máximo para altura.

min-height: 8%;

overflow


Podemos usar a propriedade overflow para tratar como o conteúdo das caixas devem se comportar quando ficam maiores que ela. Temos duas opções:

Hidden: oculta qualquer conteúdo que não caiba na caixa.

over-flow:hidden;

Scroll: adiciona uma barra de rolagem para rolar o conteúdo.

over-flow:scroll;

Auto: se for necessário, adiciona uma barra de rolagem para rolar o conteúdo.

over-flow:scroll;

Bordas


Cada caixa tem uma borda (mesmo que ela não estiver visível ou for especificada para ter 0px de largura). A borda contorna e determina a caixa. Para visualizarmos a borda precisamos definir as três propriedades principais: largura, estilo e cor.

Border-width: Define a largura de uma borda. Podemos definir uma largura para todas as bordas do elemento ou definir a largura de cada borda.

border-width: 2px;

Para definir cada lado usamos:

Border-top-width:

border-top-width: 2px;

Border-right-width:

border-rigth-width: 2px;

Border-bottom-width

border-bottom-width: 2px;

Border-left-width

border-left-width: 2px;

Estilo de Bordas


Border-style: Define o estilo de uma borda. Podemos definir um estilo para todas as bordas do elemento ou definir o estilo de cada borda.
Tipos de bordas:

Solid:

border 1px solid;

Dotted:

border 1px dotted;

Dashed:

border 1px dashed;

Double:

border 1px dolid;

Groove:

border 1px groove;

Ridge:

border 1px ridge;

Inset:

border 1px inset;

Outset:

border 1px outset;

Border-color: Define a cor de uma borda. Podemos definir uma cor para todas as bordas do elemento ou definir a cor de cada borda

border-color: red;

Preenchimento


A propriedade padding permite especificar quanto será o espaço terá entre o conteúdo e a borda do elemento. Podemos define um preenchimento para todos os lados do conteúdo ou definir o preenchimento de cada lado.

padding: 15%;

Margem


A propriedade margin define o espaço entre o elemento e os demais elementos da página. Podemos define o espaço para todos os lados da caixa ou definir o espaço de cada lado.

margin: 15%;

Centralizando


Para centralizarmos uma caixa podemos utilizar o valor auto para as margens direita e esquerda.

margin: 1% auto 1% auto;

Podemos também resumir agrupando top com bottom e right com left, fazendo assim:

margin: 1% auto;

OBS. Só podemos usar este tipo de resumo quanto quisermos que os valores top com bottom e right com left sejam iguais.

Alterar entre linha e bloco


A propriedade display permite transformar um elemento está em linha num elemento em bloco, ou vice-versa. Também pode ser usado para ocultar um elemento. Os 4 valores principais são:

inline: Isso faz com que um elemento em bloco funcione como um elemento em linha;

display: inline;

block: Isso faz com que um elemento em linha funcione como um elemento em bloco;

display: block;

Inline-block: Isso faz com que um elemento em bloco flua como um elemento em linha, mas mantendo as outras características de um elemento em bloco;

display: inline-block;

none: Isso oculta um elemento da página;

display: none;

Ocultando Caixas


Se quiser ocultar e (ou) um elemento na página sem que seu espaço seja preenchido pelo próximo, você pode utilizar a propriedade visibility. Temos dois valores para esta propriedade:

hidden: Isso oculta o elemento.

visibility:hidden;

visible: Isso exibe o elemento.

visibility:visible;

Sombra nas Caixas


A propriedade box-shadow permite adicionar uma sombra em torno da caixa :

Horizontal, Vertical, Distorção, Cor :

box-shadow: 2px 2px 10px green;

Cantos Arredondados A propriedade border-radius permite que arredondemos as bordas da caixa.

border-radius: 2px;

ou assim:

border-radius: 2px 2px 10px 20px;

Posição dos Elementos


A propriedade position define a posição do elemento em relação aos demais da página. Para configurar podemos usar as opções:

position: static: é o valor default do elemento;

position:static;

position: relative: move um elemento em relação ao local onde ele estaria no fluxo normal.

position:relative;

position: absolute: remove o elemento do fluxo normal e ele não mais afeta o fluxo dos demais elementos . Precisa usar as propriedades top, bottom, right e left para configurar.

position:absolute;

position: fixed: marca o posicionamento em relação a janela. Precisa usar as propriedades top, bottom, right e left para configurar.

position:fixed;

Sobrepondo os Elementos


Ao usar o posicionamento relativo, fixo ou absoluto os elementos podem se sobrepor. Assim, com a propriedade z-index, podemos ordenar a sequência de sobreposição. Por exemplo, um elemento com z-index de 5 ficará sobre um outro com o z-index de 4.

z-index:4;

Elementos Flutuantes


A propriedade float permite selecionar um elemento do fluxo normal e posicioná-lo o máximo possível a direita ou a esquerda do elemento container. Ao usar a propriedade float você também deve usar a propriedade width, pois você pode ter um resultado inconsistente.

float:left;

Podemos isolar elementos flutuantes utilizando a propriedade clear, ela determina que nenhum elemento deve flutuar do lado selecionado do elemento. Temos os valores: left, right, both e none.

clear:both;

Transição de Efeitos


A propriedade transition adiciona um efeito quando o navegador troca de um estilo para outro, sem usar animações em flash ou javascript. Promovendo mudanças suaves e com um tempo determinado. Temos as configurações:

transition-property: nome da propriedade que sofrerá a transição, para adicionar a todas use o valor all;

transition-property:border-color;

transition-duration: duração do efeito em segundos;

transition-duration:3s;

transition-timing-function: forma como a transição progride no tempo. Temos os valores: linear, Ease, Ease-in, Ease-out e Ease-in-out;

ease: Especifica um efeito de transição com um arranque lento, em seguida rapidamente, em seguida, termina-se lentamente (isto é o padrão).

transition-timing-function:ease;

linear: Especifica um efeito de transição com a mesma velocidade do início ao fim.

transition-timing-function:linear;

Ease-in: Especifica um efeito de transição com um arranque lento.

transition-timing-function:ease-in;

Ease-out: Especifica um efeito de transição com uma extremidade lento.

transition-timing-function:ease-out;

Ease-in-out: Especifica um efeito de transição com um lento início e fim.

transition-timing-function:ease-in-out;

Transição de Efeitos


Transition-delay: define a partir de quanto tempo em segundos que o efeito da transição vai se iniciar.

transition-delay: 0.2s;

Podemos configurar em uma só linha assim:

transition: property | duration | timing-function | delay;

transition: border-color 1s linear 0.2s;

Layout de Largura Fixa


Designs de layout de largura fixa não mudam de tamanho quando o usuário aumenta ou diminui o tamanho da janela do navegador. As medidas tendem a serem exatas e em pixels.

Como o tamanho será fixo e nosso site poderá ser visualizados em dispositivos de tamanho diferente, uma boa prática é fixar a largura do body em 960px, que é visualizado pela grande maioria e centralizá-lo deixando as margens laterais como auto.

Layout Fluído


O layout fluído usa a percentagem para especificar a largura de cada caixa de modo que o design se estenderá para se ajustar ao tamanho da tela.

Há uma boa prática de deixar o body com 90% de modo que haja um pequeno espaço entre as laterais. Use percentagem tanto para seus elementos quanto para as margens, assim seu layout não irá quebrar.

Ao testar isso no navegador, lembre-se de aumentar e reduzir a janela.

Layout Responsivo


O layout responsivo nos permite criar uma estilização própria para cada tamanho de tela, fazendo com que a página se adapte automaticamente a cada tamanho de tela.

Para trabalhar com este tipo de layout utilizamos o @media que funciona de forma dinâmica e é acionado assim que a página recebe seus atributos.

Ao testar isso no navegador, lembre-se de aumentar e reduzir a janela.
