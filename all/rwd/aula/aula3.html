Cascading Style Sheets (CSS)


É uma "folha de estilo" composta por “camadas” e utilizada para definir a apresentação (aparência) em páginas da internet que adotam para o seu desenvolvimento linguagens de marcação. O CSS define como serão exibidos os elementos contidos no código de uma página da internet e sua maior vantagem é efetuar a separação entre a apresentação e a estrutura de uma página.

Por que o CSS foi criado?


Com a evolução das páginas da internet e a exigência dos usuários, houve uma necessidade de aprimorar a forma de formatar as páginas.
Foram criados novos recursos (TAGS) no HTML, e cada vez mais ficava difícil criar e manter um padrão para as páginas.
Era necessário separar a formatação da estrutura da página. Diminuir o tempo de trabalho, aumentando a reusabilidade.
Aplicar múltiplo estilos dentro de uma única página.
Assim, o CSS foi criado junto com o HTML 4.0.

Podemos trabalhar com o CSS em um documento de 3 formas:

Folha de Estilo Incorporada: É criada uma TAG de marcação style> dentro do head do documento, onde são inseridas as instruções CSS.

Exemplo:

Codigo:

<style> h3{color:red;} </style>

Folha de Estilo Inline: É utilizado o atribulo style para inserir a estitilização diretamente dentro da TAG. Exemplo:
Codigo:

<h4 style="color:red;margin-left:2%"> Texto para visualizar o CSS</h4>

Resultado:

Folha de Estilo Externa: É criado um arquivo externo com a extensão “.css”, separando completamente a camada de estrutura da apresentação. Utilizamos a TAG < link> para conectar nosso arquivo de estilo.

Codigo:

Html:

<link rel="stylesheet" href="../../../css/styleespecifico.css"> <r>Texto para visualizar o CSS</r>

Css:

p{color: red;}

Seletores


A sintaxe padrão do CSS é bem simples, precisamos indicar um seletor que pode ser uma Tag, um Id ou uma Classe, e dentro de um bloco definido por chaves inserimos os comandos referente à formatação.

p = Seletor

{ = Bloco de instruções

color: red; = Propriedade e Valor

font-size:20pt; = Propriedade e Valor

} = Bloco de instruções

Seletores CSS


Existem vários tipos de seletores CSS, os padrões são os seletores para tags, ids e classes. Seletor para TAG: O seletor de tag é o mais simples de todos, basta informar o nome da tag, neste caso, todas as tags < p> existentes no documento receberão a formatação indicada.

Seletor para ID: O seletor para id é mais específico do que o de tags, formata somente a tag que possui o id especificado, para indicar que o seletor será para um id basta anteceder o nome do id com o caractere hash “#”.

#idDiv{border: 1px solid;}

Seletor para CLASS: Este seletor formata todos os elementos que usam a classe indicada, desta maneira podemos ter várias tags

por exemplo, somente receberão a formatação aquelas que tiverem o atributo class com o nome da classe configurada, para identificar este seletor, basta usar um ponto “.” antes do nome da classe.

.classeDiv{border: 1px solid;}

Cores


Cores em CSS podem ser especificadas pelos seguintes métodos:

ᛃ Cores hexadecimais

color: #RRGGBB

ᛃ Cores RGB

color: rgb(255, 0, 0);

ᛃ Cores RGBA

color: rgba(255, 0, 0,0.5);

ᛃ Cores HSL

color: hsl(0, 100%, 50%);

Fontes


A propriedade font-Family permite especificar a fonte que deve ser usada para qualquer texto dentro do(s) elemento(s) ao qual a regra CSS se aplica. Você pode especificar uma lista de fontes separadas por virgulas de modo que, se o usuário não tiver a primeira fonte instalada, o navegador pode tentar usar a próxima da lista.

font-family:arial;

É uma boa prática terminar a lista com o tipo da fonte principal, pois se nenhuma delas for encontrada, o navegador busca uma fonte do mesmo tipo.

Temos 3 tipos de fonte:

SERIF: ela têm detalhes adicionais nas extremidades

SANS-SERIF: ela têm as extremidades retas, sem detalhes

MONOSPACE: ela têm todas as letras com o mesmo espaçamento

@Font-face


Com o @font-face, podemos usar uma fonte existente em nosso projeto, assim não dependemos que o usuário tenha a fonte. Para isso, precisamos ter o arquivo da fonte salvo dentro do nosso projeto.

@font-face{ font-family: 'cartoon'; src:url('...fonts/cartoon.ttf') }

Font-size


A propriedade font-size permite especificar o tamanho da fonte. Podemos utilizar algumas unidades de medida para isso:
PIXELS: referente aos pontos da tela, ela permite uma precisão maior para controlar o tamanho das fontes. EX: 80px.
PERCENTAGENS: Ele altera o tamanho da fonte usando uma percentagem relativa ao seu tamanho padrão. O padrão de textos é 16px. Ex: 75% (sobre 16 seria 12px)
EMS: Um em é equivalente ao tamanho padrão web “16px”. Ex: 1.5em.

font-size:50%;

Font-weight


A propriedade font-weight deixa o texto em negrito. Temos duas variáveis desta propriedade:

font-weight: normal;

Normal: Isso faz com que o texto apareça em um peso normal. Bold: Isso faz com que o texto apareça em negrito

font-weight: bold;

Font-style


Para criar um texto em itálico usamos a propriedade font-style. Temos 3 valores possíveis para esta propriedade:

Normal: Isso faz com que o texto apareça em um estilo normal.

font-style: normal;

Italic: Isso faz com que o texto apareça em itálico.

font-style: italic;

Oblique: Isso faz com que o texto apareça obliquo.

font-style: oblique;

Textos


Text-transform


O text-transform é usado para alterar o uso de maiúsculas e minúsculas no texto. Temo os seguintes valores:

Uppercase: Isso faz com que o texto apareça em maiúsculo.

text-transform: uppercase;

Lowercase: Isso faz com que o texto apareça em minusculo.

text-transform: lowercase;

Capitalize: Isso faz com que a primeira letra de cada palavra do texto apareça em maiúsculo.

text-transform: capitalize;

Text-decoration


O text-decoration permite aplicar 5 valores diferentes:

None: Isso remove qualquer decoração já aplicada no texto.

text-decoration: none;

Underline: Adiciona uma linha abaixo do texto.

text-decoration: underline;

Overline: Adiciona uma linha acima do texto.

text-decoration: overline;

Line-through: Adiciona uma linha no meio do texto.

text-decoration: line-through;

Underline overline: Cria uma linha acima e outra abaixo do texto.

text-decoration: underline overline;

Text-align


A propriedade text-align permite controlar o alinhamento do texto. A propriedade pode receber um dos 4 valores:

left: Isso indica que o texto irá ser alinhado a esquerda.

text-aling: left;

right: Isso indica que o texto será alinhado a direita.

text-aling: right;

center: Isso faz com que o texto se alinhe ao centro.

text-aling: center;

justify: Isso faz com que o texto se alinhe dos dois dados.

text-aling: justify;

Text-shadow:


A propriedade text-shadow cria uma sobra nas letras do texto.

text-shadow: 1px 1px 1px red;

Outras configurações:


Line-height: Regula o espaço entre linhas de um parágrafo.

line-height: 20px;

Letter-spacing: Define o espaçamento entre as letras do texto.

letter-spacing: 20px;

Word-spacing: Define o espaçamento entre as palavras do texto.

word-spacing: 20px;

Text-indent: Recua a primeira linha de um objeto de texto.

text-indent: 20px;

Pseudoclasses:



Pseudoclasses (Textos)


First-letter: Ele é atribuído ao seletor do elemento e permite estilizar apenas a primeira letra do texto.

p:first-letter{font-size:20px;}

First-line: Ele é atribuído ao seletor do elemento e permite estilizar apenas a primeira linha do texto.

p:first-line{font-size:20px;}


Pseudoclasses (Links)


Temos duas pseudoclasses que utilizamos para personalizar nossos links:

Link: Isso permite definir estilos para os links que ainda não foram visitados.

p:link{color: red;}

Visited: Isso permite definir estilos para os links que já foram visitados.

p:visited{color: red;}


Pseudoclasses (Interação com o usuário)


Temos três pseudoclasses que utilizamos para interagir com os usuários:

Hover: Isso permite mudar os estilos quando o mouse passa sobre o elemento.

p:hover{color: red;}

Active: Isso permite mudar os estilos quando o usuário clica no elemento.

p:active{color: red;}

Focus: Isso permite mudar os estilos quando o elemento recebe o foco.

input[type="text"]:focus{color:blue;}


Preenchimentos do fundo


A propriedade background permite preenchermos o fundo da caixa dos elementos de diversas formas.

background-color: É a propriedade que define a cor de fundo de um elemento.

backgorund-color: #red;

background-image: É a propriedade que define uma imagem como fundo.

background-image: url('img/ola.png');


Background-repeat


A propriedade background-repeat controla a repetição de uma imagem de fundo. Podemos configurar esta propriedade de 4 formas:

background-repeat : repeat-x: imagem repete na horizontal

background-repeat:repeat-x;

background-repeat : repeat-y : imagem repete na vertical

background-repeat:repeat-y;

background-repeat : repeat: imagem repete em ambas as posições

background-repeat:repeat;

No-repeat: a imagem não se repete

background-repeat:no-repeat;

Background-attachment


background-attachment : fixed: A imagem de fundo permanece na mesma posição da página.

background-attachment: fixed;

background-attachment : scroll: A imagem de fundo move-se com o conteúdo quando o usuário rola a página.

background-attachment: scroll;

Posição do Fundo


Quando uma imagem não é repetida a propriedade background-position
especifica a posição da imagem na janela. Podemos usar os valores:

Top, Botton e Center: para alinhamento vertical;
Right, Left e Center: para alinhamento horizontal;

background-position: 50% 50%;
