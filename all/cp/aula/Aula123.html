O que é Lógica?


É a ciência que estuda as leis e critérios de validade que regem o pensamento e a demonstração, ou seja, ciência dos princípios formais do pensamento. Usar a lógica no dia a dia é fundamental, principalmente para os profissionais de TI (desenvolvedores e analistas) que possuem como princípio entender a necessidade dos usuários e trazer solução: rápida, eficiente e eficaz.



O que é Sequencia Lógica?


São passos executados até atingir um objetvo ou solução do problema.



O que é instrução?


Regra ou norma definida para realização ou emprego de algo.



O que é Algoritmo?


Sequencia finita de passos que levam a execução de uma tarefa.

Aula 2

(Apresentação do VisuAlg, tipos de dados, Expressões lógicas e Exercicios.)



VisuAlg


Esta ferramenta permite aos iniciantes em programação melhor entendimento por apresentar um ambiente próximo da realidade. Ela é capaz de simular o que acontece na tela do computador com o uso dos comandos leia e escreva, verificar os valores das variáveis, o acompanhamento passo a passo da execução do algoritmo e suporta um modo simples de depuração.



Variáveis


Variáveis são espaços reservados na memória do computador para guardar informações que serão ultizadas durante a execução de um programa.



Exemplo código VisuAlg:


Algoritmo "PROGRAMA PRESTAÇÃO"
// Disciplina : Linguagem e Lógica de Programação
// Professor : Fernanda
// Descrição : O programa calcula o novo valor a ser pago.
// Autor(a) : Bruno
// Data atual : 01/07/2018

Var
a1,a2,a3:real
resultado:real

Inicio

escreval("Informe o valor do boleto: ")
leia(a1)
escreval("Informe o valor de juros: ")
leia(a2)
escreval("Informe o numero de dias em atraso: ")
leia(a3)

resultado <= a1+(a1*(a2/100))*a3

escreva("Resultado: ",resultado)

Desvio Condicional - Simples


Quando há necessidade de executar uma instrução apenas se determinada condição acontecer.



Exemplo:


var
num: inteiro

Inicio
num < - 10

se (num > 10) entao
escreval("Maior que 10")
fimse
escreva("FIM")

Desvio Condicional - Composto


Quando há necessidade de executar uma instrução quando a resposta é SIM e quando é NÃO.



Exemplo:


var
num: inteiro

Inicio
num < - 10

se (num > 10) entao
escreval("Maior que 10")
senao
escreva("Menor ou igual a 10")
fimse
escreva("FIM")

Seleção Múltipla


Quando outras verificações são feitas caso a anterior tenha falhado. A proposta da estrutura ESCOLHA-CASO é permitir ir direto no bloco de código desejado, dependendo do valor de uma variável de verificação.



Exemplo:


algoritmo "DependentesFuncionario"
var Nome: Caractere
Sal, nSal : Real
dep : Inteiro

inicio
escreva ("Qual o nome do Funcionario?: ")
leia (Nome)
escreva ("Qual o salario do Funcionario?: R$")
leia (Sal)
escreva ("Qual e a quantidade de dependentes? ")
Leia (Dep)
escolha Dep
caso 0
nSal < - Sal + (Sal*5/100)
caso 1, 2, 3
nSal < - Sal + (Sal*10/100)
caso 4, 5, 6
nSal Sal + (Sal*15/100)
OutroCaso
nSal < - Sal + (Sal*18/100)
FimEscolha

EscrevaL ("O novo salario de ", Nome, " será de R$", nSal:5:2)
