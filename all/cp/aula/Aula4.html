Comando de entrada de dados


Normalmente precisamos de dados de entrada para serem processados pelo programa. Desta forma, precisamos de um comando para solicitar e obter dados fornecidos pelo usuário.


Scanner scn = new Scanner(System.in);

Double nota1 = scn.nextDouble();

Quando um computador encontra um comando de entrada de dados, ele suspende a execução do programa até que os dados sejam fornecidos. Para isso ultlizaremos o Scanner.



Comando de saída de dados


Da mesma forma que precisamos receber dados externos, precisamos fornecer dados e instruções para os usuários. Desta forma, precisamos de um comando para retornar dados e/ou mensagens em um dispositivo de saída.


System.out.println(var ou expressão ou mensagem);


Exemplos:


System.out.println(“O saldo atual é” + saldo);
System.out.println(soma);


import java.util.Scanner;

public class Atividade01 {

public static void main(String[] args) {

// Variáveis: nota1, nota2 e média do aluno
double nota1 = 0, nota2 = 0, media = 0;

Scanner scn = new Scanner(System.in);

System.out.println("Entre com a primeira e a segunda nota do aluno:");

// entrada de dados
nota1 = scn.nextDouble();
nota2 = scn.nextDouble();

// cálculo da média
media = (nota1 + nota2)/2;
System.out.println("A média é : "+media);
}}


Estruturas de Seleção:


• Para alterarmos o fluxo de execução de um programa escrito em Java, temos o comando if.
• Situações onde há vários caminhos de execução que dependem da avaliação de uma variável inteira, o comando switch.
• Para validar mais de uma sentença podemos utilizar o comando elseif



Comando SE:


if (condição) // onde condição pode ser uma expressão
{
Declarações ou blocos
}
else // caso a condição anterior não seja satisfeita
{
Declarações ou blocos
}
